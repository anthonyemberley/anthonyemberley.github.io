friendtract

Friendtract is an early first-stage prototype of a DApp (decentralized application) to enable smart contracts between friends. In this version, a smart contract is created between two friends who share a group in the GroupMe mobile app. The contract charges one of the friends if he/she spams the group (i.e. if he/she sends more than a set threshold number of messages per minute to the group).

This application is built using the Embark framework (https://github.com/hitchcott/meteor-embark) with Meteor. It is run using a local node on the Ethereum testrpc network. In this first attempt, real ether is not sent and smart contracts are not deployed to the blockchain. This version is instead an effort to understand how the Embark framework works, and the feasibility of a more robust application.

  • Date: Fall 2015
  • Keywords: Web Development, Ethereum, Blockchain, Meteor, Embark
  • Github: Link

Project Title

Contagion is a team game involving "smart" vests that communicate with the contagion ball via bluetooth. The object of the game is to get every member of the opposing team onto your team. The way this is done is by hitting the other team on their vest with the ball 3 times. The smart vests will begin to change color showing the team member being "contaged" by the other team, after three hits the vest turns to the other team's color and that player switches teams.


Contagion was built using bluetooth communication protocol between an Arduino in each vest, an RFduino in the ball and the player's phone which acted as the master. When the ball would come close enough to the phone (placed in a pocket on the inside of each vest) the phone would send information to the Arduino in the vest to begin coloring the vest the opposite color. The LEDs were powered by external battery packs and the application on the phone was developed in Swift.
